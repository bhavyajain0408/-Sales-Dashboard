# -Sales-Dashboard
MADHAV SALES DASHBOARD
An interactive Ecommerce Sales Dashboard developed using Microsoft Power BI Desktop to analyze sales performance, customer behavior, and profit trends across different categories, states, and payment modes.

1. Purpose
Developed an interactive Ecommerce Sales Dashboard using Microsoft Power BI Desktop to analyze sales, profit, customer trends, and payment modes. Performed data cleaning, transformation, and visualization to generate actionable business insights. Implemented KPI cards, slicers, and dynamic charts for quarter-wise and category-wise performance analysis. Strengthened skills in data visualization, dashboard development, and business intelligence reporting.

2. Project Highlights:
Designed a dynamic and interactive dashboard for ecommerce sales analysis
Performed data cleaning, transformation, and modeling for structured reporting
 * Created KPI cards to track:
> Total Sales Amount
> Quantity Sold
> Profit
> Average Order Value (AOV)
 * Built interactive visualizations including:
> Bar Charts
> Donut Charts
> Profit Trend Analysis
> Category & State-wise Analysis
 * Analyzed:
> Sales by State
> Quantity by Category
> Profit by Sub-Category
> Customer-wise Sales Performance
> Payment Mode Distribution
> Implemented filters and slicers for quarter-wise and user-driven analysis

3. 🛠 Key Technologies Used:
> Microsoft Power BI Desktop,
> DAX (Data Analysis Expressions)
> Power Query
> Data Cleaning & Transformation
> Interactive Data Visualization
> KPI Reporting & Dashboard Design
> Excel / CSV Dataset Integration
